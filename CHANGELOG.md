# Changelog

## 1.1.0

- "Copy as Markdown" context menu now appears in all webviews, not just Claude Code panels
- Scoped `Cmd+Shift+C` / `Ctrl+Shift+C` keybinding to webview contexts (webview panels, extension pages, Welcome tab) to avoid conflicting with "Open External Terminal"

## 1.0.0

- Initial release
- Converts clipboard HTML to GitHub-Flavored Markdown
- Fenced code blocks with language annotations
- GFM tables, task lists, and strikethrough
- Cross-platform clipboard support (macOS, Windows, Linux X11/Wayland)
- Keyboard shortcut `Cmd+Shift+C` / `Ctrl+Shift+C`
- Right-click context menu in webviews
